# droopy

![badge](https://github.com/kevoneil/droopy/actions/workflows/actions.yml/badge.svg)

A better, more accessible, dropdown menu

## Installation

To install this package from NPM run:

```
npm i droopy-dropdown

// or
yarn add droopy-dropdown
```
